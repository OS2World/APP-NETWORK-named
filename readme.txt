A simple name server for OS/2
=============================

This package contains a very simple domain name server for OS/2.

The package contains the following files:

README.TXT	- this file.
NAMED.DOC	- basic documentation.
NAMED.EXE	- the program executable.
NAMED.CNF	- sample configuration file
NETLIB.DLL	- a library of functions used by this and related programs.

That's all!
